# 🌐 Portfolio Website

This repository contains my **personal portfolio website**, built using **HTML, CSS, and JavaScript**. The website is fully responsive and includes multiple sections to showcase my skills, projects, achievements, certificates, and contact information.

---

## 📌 Key Features

* **Responsive Design** — Optimized for all screen sizes
* **Clean & Modern UI**
* **Home, About, Expertise, Projects, Certificates, Achievements, Resume, Contact** sections
* **Interactive Photo Gallery**
* **Fully functional contact section for inquiries**
* **Smooth animations and user-friendly navigation**

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript (ES6)**

To run the project locally:

1. **Clone the repository**

   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```

2. **Open the project folder**

   ```bash
   cd your-repo-name
   ```

3. **Launch the website**
   Open the file below in any browser:

   ```
   index.html
   ```
